Decimating filters in scipy

Posted on Mon 21 November 2016 in Python, DSP

A common DSP element is a decimating low-pass filter. If the filter is going to suppress high frequency information then there is no need to keep all the samples necessary to capture that high frequency information.

Typically in hardware we use causal FIR filters — because FIR filters lack feedback we can run them faster on the FPGA. When the filter also allows for decimation we use the polyphase technique to commute the decimation through the filter and reduce FGPA resource usage.
